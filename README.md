# RtMidiMonitor

A JavaFX port of the ShowMidi tool, utilizing the `rtmidijava` portable MIDI library.

## Prerequisites

- Java 25
- Maven

## Build and Run

To build the project:

```bash
mvn clean install
```

To run the JavaFX application:

```bash
mvn javafx:run
```

## Features (Planned)

- Real-time MIDI message monitoring.
- Portable UI using JavaFX.
- Cross-platform support via `rtmidijava`.
